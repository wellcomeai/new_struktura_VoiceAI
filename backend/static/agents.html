Укажите ID Google таблицы, в которую будут записываться все диалоги с ассистентом. Таблица должна быть доступна для редактирования по ссылке.

Инструкция по настройке:

  • Создайте новую таблицу в Google Sheets
  • Сделайте таблицу доступной для редактирования по ссылке (Настройки доступа → Изменить → Все, у кого есть ссылка)
  • Скопируйте ID из URL таблицы (часть между /d/ и следующим символом /) и вставьте выше
  • Система автоматически создаст необходимые заголовки в таблице
  • В таблицу будут записываться: время, запросы пользователя, ответы ассистента и результаты выполнения функций
