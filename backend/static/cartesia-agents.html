API ключи для Cartesia агентов

Настройте API ключи

Для работы Cartesia агентов необходимо добавить OpenAI API Key (для LLM обработки) и Cartesia API Key (для синтеза речи).

Как начать:

  1. Получите OpenAI API Key на platform.openai.com
  2. Получите Cartesia API Key на play.cartesia.ai
  3. Вставьте ключи в поля выше и нажмите "Сохранить ключи"

После добавления ключей вы сможете создавать и управлять Cartesia агентами
