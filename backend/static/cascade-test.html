Настройте API ключ

Для работы Cascade агентов необходим OpenRouter API Key. Через OpenRouter вы получаете доступ к сотням LLM моделей для обработки диалогов.

Как начать:

  1. Зарегистрируйтесь на openrouter.ai
  2. Создайте API Key в разделе ключей
  3. Вставьте ключ в поле выше и нажмите «Сохранить ключ»

После добавления ключа вы сможете создавать и управлять Cascade агентами
