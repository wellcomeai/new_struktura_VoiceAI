Настройка Google Gemini API

Для работы с Gemini агентами необходимо добавить API ключ Google AI. Это позволит вам создавать и использовать голосовых ассистентов на базе Google Gemini.

Как получить API ключ:

  1. Перейдите на Google AI Studio
  2. Войдите с вашей учетной записью Google
  3. Создайте новый API ключ (кнопка "Get API Key")
  4. Скопируйте ключ и вставьте его в поле выше
  5. Нажмите кнопку "Сохранить"

После добавления ключа вы сможете создавать и управлять Gemini агентами
