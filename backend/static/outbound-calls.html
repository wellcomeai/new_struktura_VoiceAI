Как настроить исходящие звонки:

  1. Создайте сценарий исходящих звонков в Voximplant
  2. Привяжите сценарий к правилу (Rule) и скопируйте Rule ID
  3. Выберите вашего голосового ассистента
  4. Укажите номер телефона для звонка и ваш Caller ID
  5. Нажмите "Запустить звонок"

Информация о настройке

Что происходит при запуске звонка:

  1. Отправляется HTTP запрос к Voximplant API
  2. Запускается сценарий по указанному Rule ID
  3. Сценарий запрашивает конфигурацию ассистента с вашего сервера
  4. Совершается звонок на указанный номер
  5. Голосовой ассистент ведет разговор согласно настройкам

Примечание: Убедитесь, что у вас есть активная подписка Voximplant и достаточно средств на балансе для совершения звонков.
